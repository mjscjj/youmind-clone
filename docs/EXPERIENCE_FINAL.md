## 🎯 核心功能总结

### 三栏布局
- 左侧：项目导航（250px）
- 中间：内容区域（flex）
- 右侧：AI对话面板（380px）

### 看板系统
- 项目（Board）作为组织单元
- 支持：深度研究、资料收集、灵感记录
- 新建项目、归档、活跃状态管理

### AI 技能系统（100+个）
#### 主要分类：
1. 📚 学习与阅读（深度阅读教练、论文拆解等）
2. ✍️ 写作与创作（公众号、YouTube脚本等）
3. 📊 研究与分析（深度研究Pro、麦肯锡分析等）
4. 🎨 视觉与图像（AI漫剧、3D知识卡片等）
5. 📝 摘要与总结（快速概览、播客总结等）
6. 🧠 思维与决策（纳瓦尔教练、第一性原理等）
7. 🌐 资讯与聚合（AI日报、RSS简报等）
8. 🎓 教育教学（课件设计、识字卡片等）
9. 🔧 工具与效率（Skill创建器、信息核查等）
10. 📈 可视化（思维导图、知识图谱等）

### 资料管理
- 三种类型：笔记、链接、文件
- 收集方式：直接上传、浏览器插件、iOS应用
- 支持格式：PDF、Office、音频、图片、网页、YouTube

## 💡 关键洞察

### 产品定位
"AI学习与创作智能体" - 强调学习+创作的完整闭环

### 差异化特点
1. **技能包生态** - 100+预置技能，覆盖多场景
2. **个性化洞察** - AI从用户行为中学习
3. **可编辑输出** - 所有AI生成内容可修改
4. **跨素材关联** - 发现不同内容的隐藏联系

### 核心工作流
```
素材输入 → AI处理 → 互动深化 → 输出创作
```
