## 2. 看板系统 (Boards)

### 2.1 功能概述
看板是 YouMind 的核心组织单元，用于分类管理不同类型的资料和研究项目。

### 2.2 预设看板类型
1. **深度研究** - 系统性研究项目，支持AI辅助调研
2. **我的资料** - 通用资料收集
3. **灵感收集** - 快速记录想法和灵感

### 2.3 看板操作
- **创建**: 点击"新建看板"按钮，输入名称和选择图标
- **切换**: 左侧导航栏点击切换
- **删除**: 看板设置中删除
- **重命名**: 右键或设置中修改

### 2.4 看板内组织
- 资料卡片网格/列表视图
- 拖拽排序
- 标签分类
- 搜索过滤

## 3. AI 对话系统

### 3.1 深度研究助手
- **入口**: 右侧固定面板
- **触发方式**: 
  - 直接输入问题
  - 使用 `/深度研究` 命令
  - 点击技能按钮

### 3.2 技能系统 (Skills)
| 技能名称 | 命令 | 功能描述 |
|---------|------|---------|
| 深度研究 | /深度研究 | 针对主题进行系统性调研 |
| 智能总结 | /总结 | 自动总结资料和笔记 |
| 生成报告 | /报告 | 将研究整理成结构化报告 |
| 对比分析 | /对比 | 多方案/产品对比分析 |

### 3.3 对话特性
- **思考过程展示**: 显示"思考了 X 秒"，可展开查看
- **上下文记忆**: 保持看板内对话连贯性
- **资料引用**: AI 可引用看板内的资料
- **多轮对话**: 支持追问和深入探讨

### 3.4 输入方式
- 文本输入
- 语音输入 (麦克风按钮)
- 文件上传
- 链接分享

## 4. 资料管理功能

### 4.1 三种内容类型

#### 📄 笔记 (Note)
- 富文本编辑器
- 支持 Markdown
- AI 辅助写作
- 自动保存

#### 🔗 链接 (Link)
- 自动抓取网页标题和摘要
- 生成预览卡片
- 保存网页快照
- 支持批量导入

#### 📎 文件 (File)
- 支持格式: PDF, DOC, DOCX, TXT, MD
- 自动 OCR 识别
- 内容索引和搜索
- 在线预览

### 4.2 添加方式
1. **手动添加**: 底部三个快捷按钮
2. **AI 自动收集**: 研究过程中自动保存
3. **浏览器插件**: 一键保存网页
4. **批量导入**: 支持文件夹上传

### 4.3 内容操作
- 编辑/删除
- 移动到其他看板
- 添加标签
- 设置提醒
- 分享协作

## 6. 用户流程体验

### 6.1 新用户流程
1. 注册/登录
2. 选择使用场景（个人/团队/学生/研究者）
3. 自动生成对应模板看板
4. 引导教程

### 6.2 典型使用场景

#### 场景一: 主题研究
1. 创建/选择"深度研究"看板
2. 告诉 AI 研究主题
3. AI 自动收集资料
4. 人工筛选和补充
5. AI 生成研究报告

#### 场景二: 资料收集
1. 浏览网页时遇到好文章
2. 使用浏览器插件保存链接
3. 自动分类到对应看板
4. 后续 AI 总结要点

#### 场景三: 写作辅助
1. 在看板中新建笔记
2. 使用 AI 辅助生成大纲
3. 分段写作，AI 润色
4. 引用看板内的参考资料

## 9. 产品亮点总结

1. **AI 原生设计**: 不是简单的 AI 叠加，而是深度整合
2. **研究导向**: 专注于深度研究场景，而非泛笔记工具
3. **流程闭环**: 资料收集 → 整理 → 研究 → 输出报告
4. **体验流畅**: 三栏布局，操作路径短
5. **智能助手**: 真正的研究助手，不只是聊天机器人
