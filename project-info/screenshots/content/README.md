## 所需用户操作

要完成此体验任务，需要您执行以下操作：

1. 在 Chrome 浏览器中打开 YouMind: https://youmind.com/boards
2. 点击 Chrome 工具栏中的 **OpenClaw Browser Relay** 扩展图标
3. 确保扩展图标 badge 显示为 **ON** 状态
4. 这将把当前标签页附加到 OpenClaw，使我能够进行自动化操作
