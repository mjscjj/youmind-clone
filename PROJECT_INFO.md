## 技术栈

### 前端
- React 18
- TypeScript 5
- Vite 5
- Tailwind CSS 3
- React Router 6

### 后端
- Go 1.21
- Gin Framework
- SQLite / PostgreSQL
- WebSocket

### 基础设施
- Docker
- Docker Compose

## 项目结构

```
youmind-clone/
├── src/              # 前端源代码
├── internal/         # 后端 Go 代码
├── docs/             # 项目文档
├── infra/            # Docker 配置
├── public/           # 静态资源
└── package.json      # 项目配置
```

## 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

### 构建
```bash
npm run build
```
